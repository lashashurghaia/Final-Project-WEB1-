#  ფინალური პროექტი (WEB 1)

ეს არის ფინალური პროექტი, რომელიც შექმნილია HTML5, CSS3/SCSS და JavaScript  ტექნოლოგიების გამოყენებით. პროექტი წარმოადგენს ფუნქციონალურ ვებ-გვერდს, სადაც რეალიზებულია ავტორიზაციის, რეგისტრაციის და კონტენტთან ინტერაქციის სისტემა.

# ძირითადი ფუნქციონალი

 **ავტორიზაციის სისტემა:** რეგისტრაციისა და ავტორიზაციის ფორმები, ვალიდაციითა და მონაცემების LocalStorage-ში შენახვით.
 **დინამიური ინტერფეისი:** სისტემაში შესვლის შემდეგ, ავტორიზაციის ღილაკი ავტომატურად იცვლება მომხმარებლის სახელითა და გამოსვლის (Logout) ღილაკით.
 **სიახლეების შენახვა:** მომხმარებელს შეუძლია სასურველი სიახლეები მონიშნოს და შეინახოს, რის შემდეგაც ისინი აისახება ცალკე გვერდზე (saved.html).
 **ძებნის ფუნქცია:** საიტზე ინტეგრირებულია საძიებო სისტემა კონტენტის სწრაფი ფილტრაციისთვის.
 **ადაპტიური დიზაინი:** საიტი სრულად მორგებულია სხვადასხვა ზომის ეკრანზე (Desktop, Tablet, Mobile).

 **სტილები:** CSS3, SCSS (Sass)
